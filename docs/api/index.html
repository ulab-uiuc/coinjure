API Reference

Coinjure's Python API is organized into a few key modules. Most users interact with Coinjure through the CLI, but the Python API is useful for writing custom strategies or extending the harness programmatically.

Modules

  • Strategy — Strategy ABC, StrategyContext, decision recording
  • Trading Types — Order, Trade, TradeSide, PlaceOrderResult
  • Data Sources — DataSource ABC, CompositeDataSource, event streams
  • Market Relations — MarketRelation, RelationStore, 8 relation types
  • Engine — TradingEngine, runner functions, ControlServer
